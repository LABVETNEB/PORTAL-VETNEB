# PENDIENTE DE NORMALIZACION

## TABLA CLINICS

LA TABLA CLINICS PRESENTA UNA ESTRUCTURA LEGACY NO REFLEJADA EN EL SCHEMA ACTUAL.

SE DETECTO COLUMNA:
- CLINIC_ID (NOT NULL)

ESTO DEBE AUDITARSE ANTES DE:
- ELIMINARLA
- VOLVERLA NULLABLE
- INCORPORARLA AL SCHEMA TYPESCRIPT

POR EL MOMENTO NO SE TOCA EN LA MIGRACION 0005 PARA EVITAR RIESGO SOBRE DATOS EXISTENTES.
